# 路线图

路线图描述的是发布目标，不等同于完成承诺。具体任务、验收条件和优先级见根目录 [TODO](../TODO.md)。

| 阶段 | 目标 | 核心交付 |
| --- | --- | --- |
| v0.1.0-alpha.1 | 可联机验证的基础版本 | Remote 独立语音服务器、自动鉴权、设备设置、范围/队伍语音、基础方向感、Release 自动构建。 |
| 后续 Alpha | 稳定性收尾 | 双客户端回归、设备热插拔、状态诊断、范围和队伍生命周期测试、Windows 单元测试恢复。 |
| Beta 音频 | 可用音频链路 | Opus、抖动缓冲、丢包处理、语音激活、持久化音量控制。 |
| Beta 空间 | 可解释的空间效果 | 服务端权威位置、方块样本契约、距离衰减、基础遮挡和空间化平滑。 |
| RC | 运维与兼容性 | 管理命令、限流、指标、配置迁移、兼容矩阵和发布验收记录。 |
| v1.0 | 社区可用版本 | 在明确人数与网络条件下完成稳定性验证，并公布支持范围与已知限制。 |

以下能力不属于当前 Alpha：内置 Local 服务器、跨服语音、录音/回放、面向大规模正式服的性能和抗攻击承诺。
